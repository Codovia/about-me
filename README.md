# Simple Resume Website

A beginner-friendly resume and bio-data website built with HTML, CSS, and jQuery.

## Pages

- **Home** - Welcome page with introduction
- **Resume** - Education, skills, and experience
- **Bio-Data** - Personal information and contact form

## Features

- Simple navigation menu
- Animated skill bars
- Contact form with validation
- Hover effects on buttons and cards
- Responsive design
- Back to top button

## Technologies

- HTML5
- CSS3
- jQuery
- JavaScript

## How to Run Locally

```bash
cd /home/nandan/Desktop/web_assignment/one
python3 -m http.server 8000
```

Then open: http://localhost:8000

## Deploy to Netlify

**Easiest Method:**
1. Go to https://app.netlify.com/drop
2. Drag the entire folder into the browser
3. Done! Your site is live.

## Files

- `index.html` - Home page
- `resume.html` - Resume page
- `biodata.html` - Bio-data page
- `style.css` - All styles
- `script.js` - jQuery code

## Customization

Replace placeholder content with your own:
- Name and title
- Education details
- Skills and percentages
- Work experience
- Personal information
- Contact details
